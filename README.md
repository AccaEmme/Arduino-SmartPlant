# Arduino-SmartPlant
Hygrometer detector for plants with LCD and servo motor

 - SmartPlant.ino -> Arduino source code
 - SmartPlant.fzz -> Fritzing schema
 - SmartPlant.png -> Image schema (by fritzing)
